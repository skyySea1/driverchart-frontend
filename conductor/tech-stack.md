# Technology Stack

## Frontend Core
- **Framework:** Vue 3 (Composition API)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, PostCSS

## State & Data
- **State Management:** Pinia
- **Validation:** Zod
- **API Client:** Axios
- **Backend/API:** Fastify (with OpenAPI/Swagger), Cloud Functions
- **Database/Auth:** Firebase (Firestore, Authentication)

## Utilities & Libraries
- **Date Handling:** Day.js
- **Icons:** Lucide Vue Next
- **Environment:** Dotenv
